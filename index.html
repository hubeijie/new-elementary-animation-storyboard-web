方式 A：浏览器调用 API

可选 Gemini、OpenAI、Claude，或 OpenAI 兼容（国内/自建）。海外官方接口常被浏览器 CORS 拦截；国内兼容地址是否允许浏览器直连以各平台说明为准，失败请用「复制提示词」或本地 Python。
